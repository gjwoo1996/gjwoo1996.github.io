# 설정 파일 (hugo.toml)

`hugo.toml`은 블로그의 중앙 설정 파일입니다.  
블로그 제목, 메뉴 구성, 기능 ON/OFF 등 모든 설정이 이 파일에 있습니다.

---

## 전체 설정 항목 설명

```toml
baseURL = "https://gjwoo1996.github.io/"  # 블로그 주소
languageCode = "ko-KR"                    # 언어 설정 (한국어)
title = "gjwoo1996's blog"               # 브라우저 탭에 보이는 제목
theme = "PaperMod"                        # 사용할 테마 이름
```

---

### [pagination] — 페이지네이션 설정

```toml
[pagination]
  pagerSize = 10    # 목록 페이지당 글 개수 (기본값: 10)
```

---

### [params] — 블로그 세부 기능 설정

```toml
[params]
  env = "production"        # 운영 환경 (RSS, SEO, Analytics 등 활성화)
  description = "개발 공부 기록"  # 검색엔진(구글)에 표시될 설명
  author = "gjwoo1996"     # 글쓴이 이름
  defaultTheme = "auto"    # 다크/라이트 모드
                            #   "auto"  = 시스템 설정 따름
                            #   "dark"  = 항상 다크
                            #   "light" = 항상 라이트
  ShowReadingTime = true   # 글 상단에 "읽는 데 N분" 표시
  ShowShareButtons = false # 공유 버튼 표시 여부
  ShowPostNavLinks = true  # 글 하단에 이전글/다음글 버튼
  ShowBreadCrumbs = true   # 상단 경로 표시 (홈 > Posts > 글제목)
  ShowCodeCopyButtons = true  # 코드 블록 오른쪽 위에 복사 버튼
  ShowToc = true           # 목차(Table of Contents) 표시
  TocOpen = false          # 목차 기본적으로 접어두기 (false = 접힘)
```

---

### [params.homeInfoParams] — 메인 페이지 환영 문구

```toml
[params.homeInfoParams]
  Title = "안녕하세요 👋"
  Content = "개발 공부와 일상을 기록하는 블로그입니다."
```

메인 페이지(`/`) 접속 시 상단에 표시되는 소개 글입니다.

---

### [[params.socialIcons]] — SNS 아이콘

```toml
[[params.socialIcons]]
  name = "github"
  url = "https://github.com/gjwoo1996"

[[params.socialIcons]]
  name = "email"
  url = "mailto:gjwoo9602@gmail.com"
```

메인 페이지에 아이콘으로 표시됩니다. `[[]]` 형태로 여러 개 추가할 수 있습니다.

사용 가능한 아이콘 이름 예시: `github`, `twitter`, `linkedin`, `instagram`, `email`, `rss`

### [menu] — 상단 네비게이션 메뉴

```toml
[menu]
  [[menu.main]]
    identifier = "posts"  # 내부 식별자 (중복 없어야 함)
    name = "Posts"        # 화면에 보이는 이름
    url = "/posts/"       # 클릭 시 이동할 경로
    weight = 10           # 숫자가 작을수록 왼쪽에 배치

  [[menu.main]]
    identifier = "tags"
    name = "Tags"
    url = "/tags/"
    weight = 20

  [[menu.main]]
    identifier = "search"
    name = "Search"
    url = "/search/"
    weight = 30
```

새 메뉴를 추가하려면 `[[menu.main]]` 블록을 복사해서 `weight`를 조정합니다.

---

### [outputs] — 출력 형식

```toml
[outputs]
  home = ["HTML", "RSS", "JSON"]
```

- `HTML`: 웹 페이지
- `RSS`: RSS 피드 (`/index.xml`)
- `JSON`: 검색 기능에 필요한 데이터 파일 (`/index.json`)

**JSON을 제거하면 검색 기능이 동작하지 않습니다.**

---

### [markup.highlight] — 코드 블록 스타일

```toml
[markup]
  [markup.highlight]
    noClasses = false          # 외부 CSS로 코드 색상 제어 (false = CSS 파일 사용)
    lineNos = true             # 코드 블록에 줄 번호 표시
    lineNumbersInTable = true  # 줄 번호를 테이블 형식으로
                               # (복사 시 줄 번호가 포함되지 않음)
```
