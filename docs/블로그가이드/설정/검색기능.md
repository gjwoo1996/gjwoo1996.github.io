# 검색 기능

## 동작 원리

이 블로그의 검색은 **클라이언트 사이드 검색**입니다.  
서버 없이 브라우저에서 직접 검색이 이루어집니다.

```
Hugo 빌드 시 → 모든 글의 내용을 담은 index.json 파일 생성
        ↓
사용자가 /search/ 페이지 접속
        ↓
브라우저가 index.json 다운로드
        ↓
검색어 입력 시 index.json에서 일치하는 글 찾기
```

---

## 검색 기능 활성화에 필요한 세 가지

### 1. `hugo.toml`에 JSON 출력 설정

```toml
[outputs]
  home = ["HTML", "RSS", "JSON"]
```

이 설정이 없으면 `index.json`이 생성되지 않아 검색이 동작하지 않습니다.

### 2. `content/search/index.md` 파일

```yaml
---
title: "Search"
layout: "search"
summary: "search"
placeholder: "검색어를 입력하세요"
---
```

`layout: "search"` 가 PaperMod의 검색 레이아웃을 사용하게 합니다.  
이 파일은 한 번 만들면 수정할 일이 없습니다.

### 3. `hugo.toml`에 검색 메뉴 등록

```toml
[[menu.main]]
  identifier = "search"
  name = "Search"
  url = "/search/"
  weight = 30
```

---

## 검색 페이지 접근

상단 메뉴의 **Search**를 클릭하거나 `/search/` 주소로 직접 접근합니다.

검색어를 입력하면 제목, 내용, 태그 등을 기준으로 관련 글이 실시간으로 표시됩니다.
