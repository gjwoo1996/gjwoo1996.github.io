# 글 작성 방법

## 1. Front Matter (글 메타데이터)

모든 글은 맨 위에 `---`로 감싼 **Front Matter**가 있어야 합니다.  
글의 제목, 날짜, 태그 등 기본 정보를 여기에 작성합니다.

```yaml
---
title: "글 제목"                          # 필수
date: 2026-05-03T10:00:00+09:00          # 필수 (작성일, +09:00은 한국 시간대)
draft: false                              # true면 로컬에서만 보임, false면 실제 배포
tags: ["태그1", "태그2"]                  # 태그 목록 (없어도 됨)
categories: ["카테고리"]                  # 카테고리 (없어도 됨)
description: "글 요약"                    # 목록 페이지에 표시되는 설명
---
```

### draft 옵션 활용

| 값 | 동작 |
|----|------|
| `draft: true` | 로컬(`hugo server -D`)에서만 보임. 배포 시 제외됨 |
| `draft: false` | 배포 시 실제로 공개됨 |

글을 완성하기 전에 `draft: true`로 작업하다가 완성되면 `false`로 바꿔서 push합니다.

## 2. Markdown 문법

### 제목

```markdown
# 제목1 (H1) — 페이지 제목, 글 안에서는 잘 사용 안 함
## 제목2 (H2) — 주요 섹션 구분
### 제목3 (H3) — 하위 섹션
#### 제목4 (H4) — 세부 항목
```

### 텍스트 강조

```markdown
**굵게**
*기울임*
~~취소선~~
`인라인 코드`
```

### 목록

```markdown
- 순서 없는 목록
- 항목2
  - 하위 항목 (들여쓰기 2칸)

1. 순서 있는 목록
2. 항목2
3. 항목3
```

### 링크와 이미지

```markdown
[링크 텍스트](https://example.com)

![이미지 설명](/images/photo.png)
```

### 인용문

```markdown
> 이것은 인용문입니다.
> 여러 줄도 가능합니다.
```

### 코드 블록

````markdown
```python
print("Hello, World!")
```

```bash
git add .
git commit -m "메시지"
```
````

언어 이름을 명시하면 문법 강조(syntax highlighting)가 적용됩니다.

### 표(Table)

```markdown
| 열1 | 열2 | 열3 |
|-----|-----|-----|
| 내용1 | 내용2 | 내용3 |
| 내용4 | 내용5 | 내용6 |
```

### 구분선

```markdown
---
```

## 3. 새 글 생성하는 두 가지 방법

### 방법 A: Hugo 명령어로 생성 (권장)

```bash
# 프로젝트 폴더에서 실행
hugo new posts/글파일명.md
```

`archetypes/default.md` 템플릿이 자동으로 적용됩니다.  
`hugo new posts/2026-05-03-spring-boot-study.md` 처럼 날짜를 파일명에 포함하면 관리하기 편합니다.

### 방법 B: 파일 직접 생성

`content/posts/` 안에 `.md` 파일을 만들고 Front Matter를 직접 작성합니다.

## 4. 로컬에서 미리보기

```bash
# draft 포함해서 로컬 서버 실행
hugo server -D

# 브라우저에서 http://localhost:1313 접속
```

파일을 저장할 때마다 브라우저가 자동으로 새로고침됩니다.

서버를 종료하려면 터미널에서 `Ctrl + C`를 누릅니다.

## 5. 태그와 카테고리 활용

### 태그 vs 카테고리 구분

- **tags**: 글의 세부 키워드 (`"Python"`, `"Django"`, `"버그픽스"`, `"알고리즘"`)
- **categories**: 큰 분류 (`"개발"`, `"일상"`, `"독서"`, `"프로젝트"`)

### 카테고리 메뉴 추가

현재 설정에서는 `/tags/` 페이지만 메뉴에 있습니다.  
categories도 메뉴에 추가하려면 `hugo.toml`에 아래를 추가합니다:

```toml
[[menu.main]]
  identifier = "categories"
  name = "Categories"
  url = "/categories/"
  weight = 25
```
